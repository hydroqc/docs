## Enable debugging


Debug can be enabled through --log-level or --http_log_level
It can also be set through env variables :

```
HQ2M_CONTRACTS_0_LOG_LEVEL=DEBUG 
HQ2M_CONTRACTS_0_HTTP_LOG_LEVEL=DEBUG
```
